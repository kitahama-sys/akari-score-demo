```bash
cd /home/ubuntu/akari-score-demo

```

```

```

## 🛠️ トラブルシューティング

### サーバーが起動しない場合

1. MySQLサービスが起動しているか確認
   ```bash
   sudo service mysql status
   ```

2. ポート3000が使用中でないか確認
   ```bash
   netstat -tuln | grep 3000
   ```

3. 依存関係を再インストール
   ```bash
   cd /home/ubuntu/akari-score-demo
   pnpm install
   ```

### データベース接続エラーの場合

1. MySQLの接続情報を確認
   ```bash
   mysql -u root -ppassword -e "SHOW DATABASES;" | grep akari
   ```

2. データベースが存在しない場合は再作成
   ```bash
   mysql -u root -ppassword -e "CREATE DATABASE IF NOT EXISTS akari_score_demo;"
   mysql -u root -ppassword akari_score_demo < create-demo-schema.sql
   node seed-evaluation-data-demo.mjs
   node seed-demo-users.mjs
   ```
