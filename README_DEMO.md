## ✨ 主な特徴

### 🏢 デモ介護サービス専用MVV
- ビジョン: 地域に根ざした、心温まる介護を
- ミッション: 笑顔をつなぐ、未来をつくる。
- バリュー: 思いやりの心（8つの行動指針）

### 📊 評価システム
- **公開評価項目:** 4項目
  - 挨拶・身だしなみ
  - 報告・連絡・相談
  - チームワーク
  - 介護技術
- **非公開項目:** 36項目（製品版でのみ閲覧可能）

- 個別のキャリアマップ

- 介護サービス向けデザイン

## 📁 プロジェクト構成

```
akari-score-demo/
├── client/              # フロントエンド (React + TypeScript)
│   └── src/
│       ├── pages/       # ページコンポーネント
│       └── components/  # 共通コンポーネント
├── server/              # バックエンド (Express + tRPC)
├── drizzle/             # データベーススキーマ
├── *.mjs                # データ投入スクリプト
├── *.sql                # SQLスクリプト
└── docs/                # ドキュメント
```

## 🛠️ 技術スタック

- **フロントエンド:** React, TypeScript, Vite, TailwindCSS
- **バックエンド:** Node.js, Express, tRPC
- **データベース:** MySQL
- **ORM:** Drizzle ORM
- **認証:** パスワード認証（bcrypt）

## 🎮 デモシナリオ

### シナリオ1: 一般ユーザー体験
1. demo-user1でログイン
2. ダッシュボードで評価結果を確認

1. demo-adminでログイン

### シナリオ3: 成長の軌跡
1. 評価履歴で3期間の推移を確認
2. レーダーチャートで可視化
3. 評価比較機能を使用

## 📊 データベース情報

**データベース名:** akari_score_demo

**テーブル:**
- users (ユーザー)
- categories (カテゴリ)

- evaluationScores (評価スコア)
- roadmaps (キャリアマップ)
- roadmapSteps (マップステップ)
- roadmapGoals (目標)
- activityLogs (活動ログ)

## 🔄 データの再投入

### 評価項目のみ再投入
```bash
node seed-evaluation-data-demo.mjs
```

### ユーザーとサンプルデータの再投入
```bash
node seed-demo-users.mjs
```

### 完全リセット
```bash
mysql -u root -ppassword akari_score_demo < create-demo-schema.sql
node seed-evaluation-data-demo.mjs
node seed-demo-users.mjs
```
