# 灯SCORE 管理者用マニュアル

**株式会社PLACE LIBRE 社内用**  
**作成日：2025年11月14日**  
**対象：管理者・上長**

---

## はじめに

このマニュアルは、**灯SCORE（あかりスコア）** の管理者・上長向けの操作ガイドです。看護師メンバーの成長を温かく見守り、評価を通じて一人ひとりの灯を増やしていくためのシステムです。

**灯SCOREでできること：**

- メンバーの自己評価を確認する
- 上長評価を入力する
- メンバーのキャリアマップ（ロードマップ）を確認・更新する
- チーム全体の成長を可視化する

## 1. ログイン方法

### 手順

1. **ブラウザで灯SCOREにアクセス**
   - URLを開く：`https://akariscor-8ngfxyzb.manus.space`
   - Google ChromeまたはMicrosoft Edgeを推奨します

2. **ログイン画面が表示される**
   - 「ユーザー名」と「パスワード」を入力してください
   - 管理者の場合は、管理者アカウントでログインしてください

3. **「ログイン」ボタンをクリック**
   - ダッシュボード画面が表示されます

**💡 ポイント：**
- パスワードを忘れた場合は、システム管理者に連絡してください
- 初回ログイン時は、パスワードを変更することをおすすめします

---

## 2. ダッシュボードの見方

ログイン後、最初に表示される画面が**ダッシュボード**です。ここでは、チーム全体の状況を一目で確認できます。

### ダッシュボードの構成

ダッシュボードには、以下の情報が表示されます：

| セクション | 説明 |
|-----------|------|
| **評価期間サマリー** | 現在の評価期間と、自己評価・上長評価の提出状況 |
| **評価待ちメンバー** | 上長評価が必要なメンバーの一覧 |
| **最近の活動** | 自己評価提出、上長評価完了、ロードマップ更新などの履歴 |

**💡 ポイント：**
- 「評価待ちメンバー」に表示されているメンバーは、早めに評価を入力しましょう
- 「最近の活動」で、チーム全体の動きを把握できます

## 3. 評価期間の管理

灯SCOREでは、**評価期間**ごとにメンバーの成長を記録します。評価期間は、通常3ヶ月または6ヶ月単位で設定します。

### 現在の評価期間を確認する

ダッシュボードの「評価期間サマリー」セクションで、現在の評価期間を確認できます。

**表示例：**
```
2025年4月期（2025年4月1日 〜 2025年6月30日）
自己評価提出: 8/10人
上長評価完了: 5/10人
```

### 評価期間の状態

評価期間には、以下の3つの状態があります：

| 状態 | 説明 |
|------|------|
| **予定** | まだ開始していない評価期間 |
| **進行中** | 現在進行中の評価期間（メンバーが自己評価を入力できる） |
| **完了** | 終了した評価期間（編集不可） |

**💡 ポイント：**
- 評価期間が「進行中」の間に、メンバーは自己評価を入力します
- 上長は、自己評価が提出された後に上長評価を入力します

## 4. 上長評価の入力方法

メンバーが自己評価を提出したら、上長評価を入力します。

### 手順

1. **左側のメニューから「評価管理」をクリック**
   - 評価管理画面が表示されます

2. **評価期間を選択**
   - 画面上部のドロップダウンから、評価期間を選択します
   - 例：「2025年4月期」

3. **評価するメンバーを選択**
   - メンバー一覧から、評価するメンバーの「評価する」ボタンをクリックします

4. **自己評価を確認**
   - メンバーが入力した自己評価が表示されます
   - 各項目の点数とコメントを確認してください

5. **上長評価を入力**
   - 各評価項目に対して、1〜5点で評価を入力します
   - 「上長コメント」欄に、温かいフィードバックを入力してください

6. **「保存」ボタンをクリック**
   - 上長評価が保存されます
   - メンバーは、上長評価を確認できるようになります

### 評価項目

灯SCOREでは、以下の6つの項目で評価を行います：

| 項目 | 説明 |
|------|------|
| **専門知識** | 看護の専門知識や技術の習得度 |
| **コミュニケーション** | 患者さんやチームメンバーとのコミュニケーション能力 |
| **チームワーク** | チームへの貢献度や協調性 |
| **問題解決** | 課題を発見し、解決する能力 |
| **リーダーシップ** | チームをまとめ、導く能力 |
| **成長意欲** | 学び続ける姿勢や向上心 |

**💡 ポイント：**
- 評価は、メンバーの成長を促すための温かいフィードバックです
- 点数だけでなく、コメントで具体的な行動や改善点を伝えましょう
- 「ここが良かった」「次はこうしてみよう」という形で、前向きなメッセージを心がけてください

## 5. メンバーのキャリアマップを確認する

灯SCOREでは、メンバー一人ひとりの**キャリアマップ（ロードマップ）**を管理できます。キャリアマップは、メンバーの目標や成長の道筋を可視化したものです。

### 手順

1. **左側のメニューから「メンバー管理」をクリック**
   - メンバー一覧が表示されます

2. **確認したいメンバーの「詳細」ボタンをクリック**
   - メンバーの詳細画面が表示されます

3. **「キャリアマップ」タブをクリック**
   - メンバーのキャリアマップが表示されます

### キャリアマップの見方

キャリアマップには、以下の情報が表示されます：

- **現在の役割**：メンバーの現在の役職や担当業務
- **短期目標**：3〜6ヶ月以内に達成したい目標
- **中期目標**：1〜2年以内に達成したい目標
- **長期目標**：3〜5年以内に達成したい目標
- **必要なスキル**：目標達成に必要なスキルや資格

**💡 ポイント：**
- キャリアマップは、メンバーと一緒に作成・更新します
- 定期的に見直し、目標の進捗を確認しましょう

## 6. 新しいメンバーを追加する

新しいメンバーが入社したら、灯SCOREにアカウントを追加します。

### 手順

1. **左側のメニューから「メンバー管理」をクリック**
   - メンバー一覧が表示されます

2. **「新しいメンバーを追加」ボタンをクリック**
   - メンバー追加画面が表示されます

3. **メンバー情報を入力**
   - **ユーザー名**：ログイン時に使用するユーザー名（例：`tanaka_hanako`）
   - **名前**：メンバーの氏名（例：`田中 花子`）
   - **パスワード**：初回ログイン用のパスワード（メンバーに伝えてください）
   - **役割**：「メンバー」を選択（管理者の場合は「管理者」を選択）

4. **「保存」ボタンをクリック**
   - 新しいメンバーが追加されます
   - メンバーに、ユーザー名とパスワードを伝えてください

**💡 ポイント：**
- 初回ログイン後、メンバーにパスワードを変更してもらいましょう
- ユーザー名は、メンバーが覚えやすいものにしてください

## 7. 評価期間を追加する

新しい評価期間を追加する方法です。通常、3ヶ月または6ヶ月ごとに新しい評価期間を作成します。

### 手順

1. **左側のメニューから「評価期間管理」をクリック**
   - 評価期間一覧が表示されます

2. **「新しい評価期間を追加」ボタンをクリック**
   - 評価期間追加画面が表示されます

3. **評価期間情報を入力**
   - **期間名**：評価期間の名前（例：`2025年7月期`）
   - **開始日**：評価期間の開始日（例：`2025年7月1日`）
   - **終了日**：評価期間の終了日（例：`2025年9月30日`）

4. **「保存」ボタンをクリック**
   - 新しい評価期間が追加されます
   - 状態は「予定」になります

5. **評価期間を開始する**
   - 開始日になったら、「開始」ボタンをクリックして評価期間を開始します
   - 状態が「進行中」に変わり、メンバーが自己評価を入力できるようになります

**💡 ポイント：**
- 評価期間は、事前に計画して追加しておきましょう
- 評価期間が重複しないように注意してください

## 8. よくある質問

### Q1. メンバーが自己評価を提出したか確認するには？

**A1.** ダッシュボードの「評価期間サマリー」セクションで、自己評価の提出状況を確認できます。また、「評価管理」画面で、メンバーごとの提出状況を確認できます。

---

### Q2. 上長評価を修正したい場合は？

**A2.** 評価期間が「進行中」の間は、上長評価を修正できます。「評価管理」画面から、該当するメンバーの「評価する」ボタンをクリックし、評価を修正してください。評価期間が「完了」になると、修正できなくなります。

---

### Q3. メンバーのパスワードを忘れた場合は？

**A3.** 管理者は、「メンバー管理」画面から、メンバーのパスワードをリセットできます。該当するメンバーの「詳細」ボタンをクリックし、「パスワードをリセット」ボタンをクリックしてください。新しいパスワードをメンバーに伝えてください。

---

### Q4. 評価期間を削除したい場合は？

**A4.** 評価期間が「予定」の状態であれば、削除できます。「評価期間管理」画面から、該当する評価期間の「削除」ボタンをクリックしてください。「進行中」または「完了」の評価期間は削除できません。

---

### Q5. キャリアマップを更新するには？

**A5.** 「メンバー管理」画面から、該当するメンバーの「詳細」ボタンをクリックし、「キャリアマップ」タブを開きます。「編集」ボタンをクリックして、キャリアマップを更新してください。
